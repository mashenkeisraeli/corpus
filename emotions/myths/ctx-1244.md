## Original translation

Er begann, in Kumarbis Innerem zu sprechen:
„[ ... ] die stehenden [ ... ] den Ort [ ... ]
Wenn ich [ ... ],
zerquetscht [ ... ] die wie Rohr.
Wen[n] ich [aus] dem guten Ort [he]rauskomme,
wird es mich deswegen auch verunreinigen.
Im Himmel (und) auf der Erde [ ... ]
wird es mich mit dem Ohr verunreinigen.
Wenn ich aus dem guten Ort aber herauskomme,
schreit es mir [ ... ] w[ie] eine Frau [ ... ]
Als ich den Wettergott des Himmels hine[in- ... ],
bestimmte ich es.
Wie einen Stein zerbrach er ihn, seinen Kopf, den Kumarbi,
und diesem oben aus seinem Kopf kam er, KA.ZAL, der heldenhafte König.“
