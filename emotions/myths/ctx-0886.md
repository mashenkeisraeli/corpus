## Original Translation

[ ... ] des [Zorn]igen Seele [ ... ] erstickte [im Feuer]holz.
[Wen]n man [Feuerholz] in Brand se[t]zt,
wird es [in den vier Eck]en hell.
Das Kind der [St]erblichkeit machte [ ... ] die [W]orte.

## English Translation

... the angry soul ... suffocated in the firewood.When you set firewood on fire, it becomes light in the four corners. The Child of Mortality made... the words.... should also become light in it.In the same way, anger, wrath, iniquity (and) anger should also burn away.
