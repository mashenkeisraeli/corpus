# Hittite Emotions in Context

The dataset presents emotionally rich contexts from the Hittite texts.

The “Akkadian and Hittite Emotions in Context (AHEC) – Towards a Lexicon of Emotions in Cuneiform Texts from Ancient Mesopotamia and Hittite Anatolia”
is a collaboration between Johannes Gutenberg-Universität Mainz and University of Tel Aviv, funded by DFG. 

Project members: Prof. Doris Prechel, Prof. Ulrike Steinert, Prof. Amir Gilan, Maria Molina, Letizia Savino
