Welche Ursache ich durch das Orakel ermittelt habe
[und] welche Gründe für die Seuche festgestellt worden sind,

Vor [ ... ] habe ich [ ... ] geopfert.
[ ... D]IR, dem Wettergott von Ḫatti, weg[en ].
Und das Ritual [ ... ] euch/ihnen [ ... ].
[W]a[s] das betrifft, dass das Rit[ual f]ür den Fluss Māla aber (als eine Ursache) fü[r ] festgestellt worden ist,
und was das betrifft, dass ich hie[r] (gerade) [zum Fluss M]āla gehe:
Wettergott von Ḫa[t]ti, mein Herr, (und) ihr (anderen) Götter, meine Herren, lasst mir das Ritual für den Fluss Mā[l]a zu!
